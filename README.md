Простой ИИ для игры в крестики нолики , который использует алгоритм minimax , Он выбирает оптимальный ход для текущего игрока, пытаясь максимизировать свой результат (ИИ) и минимизировать результат игрока . Этот алгоритм использует рекурсию для просмотра всех возможных ходов и выбора наиболее выгодного.
